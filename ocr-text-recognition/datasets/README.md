NIA 학습 데이터는 현재 폴더에 아래의 구성으로 정리되어 있어야 합니다.

````text

|- datasets/
    |- handwritten/ : 손글씨 데이터셋 
        |- images/  : 손글씨 데이터 이미지 폴더 
            |- XXXXXXXX.png
            |- XXXXXXXX.png
            |- ...
        |- dataset_info.json : 손글씨 라벨 파일
    |- printed/     : 인쇄체 데이터셋
        |- images/  : 인쇄체 이미지 폴더 
            |- XXXXXXXX.png
            |- XXXXXXXX.png
            |- ...        
        |- dataset_info.json : 인쇄체 라벨 파일

````
